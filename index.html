Projects

Smart Shopping Trolley With Automated Billing System:

This project implements an RFID-enabled shopping cart for an improved customer experience, leveraging an Arduino for central processing. Here's the technical breakdown:

  • RFID Tags: Each product in the store is affixed with a unique RFID tag.
  • RFID Reader: The shopping cart is equipped with an RFID reader that detects nearby tags.
  • Arduino Uno (or compatible): This open-source microcontroller board interprets RFID data and manages the system's logic.
  • LCD Display: A user-friendly LCD screen displays product information (name, price) upon detection.
  • Billing System: The Arduino code maintains a running total of scanned items, generating a real-time bill.
